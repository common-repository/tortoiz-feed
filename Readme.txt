== Description ==

Tortoiz Feed is a free premium quality plugin which can show instagram feed with many custom designs.<br/>

You can achieve following with our Plugin:-<br/>

1. You can set the width and height of your feed.
2. Select number of photos to be displayed on your feed.
3. Background color can be set to match with your website design.
4. Number of columns for your feed can be set.
5. Disable on Mobile function.
6. Follow me button for instagram can be turned on or off.
7. Change background color of Follow me button.
8. Change Text color of Follow me button.
9. Custom Text on Follow me button.
10. Display feed anywhere using Shortcode.
